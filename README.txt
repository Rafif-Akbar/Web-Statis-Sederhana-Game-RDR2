# 🤠 Red Dead Redemption 2 - Static Website

Selamat datang di repositori website statis **Red Dead Redemption 2 (RDR2)**! Website ini dibuat sebagai tribute/fan-page untuk salah satu game *open-world* terbaik besutan Rockstar Games. Proyek ini dibangun menggunakan HTML dan CSS murni.

## 🌟 Fitur Utama

Website ini terdiri dari 3 menu utama yang menyajikan informasi komprehensif mengenai game Red Dead Redemption 2:

### 1. 🏠 Home
Halaman beranda yang menampilkan profil singkat dari 4 karakter sentral dalam cerita RDR2:
*   **Arthur Morgan:** Sang protagonis utama dan penegak hukum andalan geng Van der Linde.
*   **John Marston:** Karakter ikonik yang berjuang mencari kehidupan damai untuk keluarganya.
*   **Dutch van der Linde:** Pemimpin karismatik namun kompleks dari geng Van der Linde.
*   **Hosea Matthews:** Tangan kanan Dutch yang bijak dan ahli strategi.

### 2. 📜 Info
Halaman informatif yang mengupas tuntas tentang game ini, meliputi:
*   **Sinopsis:** Latar belakang cerita jatuhnya era *Wild West* pada tahun 1899.
*   **Karakter Utama:** Penjelasan lebih dalam mengenai dinamika geng Van der Linde.
*   **Gameplay:** Informasi mengenai mekanik permainan, mulai dari tembak-menembak, berburu, hingga interaksi dengan NPC dan dunia yang dinamis.

### 3. 🎬 Trailer
Halaman khusus yang didedikasikan untuk menampilkan cuplikan visual epik dari game. Menampilkan *embedded video* dari trailer resmi Red Dead Redemption 2 agar pengunjung dapat merasakan langsung atmosfer permainan.

---

## 🛠️ Teknologi yang Digunakan

*   **HTML5:** Untuk struktur dasar dan layout website.
*   **CSS3:** Untuk styling, pewarnaan, dan tata letak responsif.

---

## 🚀 Cara Menjalankan Secara Lokal

Karena ini adalah website statis, Anda tidak perlu menginstal *server* atau *dependency* yang rumit. Ikuti langkah berikut:

1.  **Clone repositori ini:**
```bash
    git clone [https://github.com/Rafif-Akbar/Web-Statis-Sederhana-Game-RDR2.git](https://github.com/Rafif-Akbar/Web-Statis-Sederhana-Game-RDR2.git)
    ```
2.  Buka folder proyek yang baru saja di-clone.
3.  Klik ganda pada file `index.html` untuk membukanya di browser default Anda (Chrome, Firefox, Safari, dll).
